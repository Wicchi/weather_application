Please insert the values above to update the data and refresh the map which shows Prague as a default. Make sure that you are entering correct longitude and latitude values.

Once you hit Submit, the current weather information for this location will be displayed at the top bar.

The map will also focus on the location you entered, and display the coordinates when clicked on the point.

A forecast chart with historical data for 2m temperature will be displayed below for the location you entered as well.

The data shown in this app is fetched using Open-Meteo API and using Open Layers for the map and Chart.js for the chart.
